Traffic Modes

Stress

High-volume stress test using TCP SYN, UDP, ICMP, and HTTP traffic simultaneously at maximum rate.

  • TCP SYN Flood
  • UDP Flood
  • ICMP Flood
  • HTTP Flood
Use this mode
Scan

Port scanning simulation using SYN, FIN, and connect scans to test IDS/IPS detection.

  • SYN Scan
  • FIN Scan
  • Full Port Scan
Use this mode
Mixed

Realistic mixed traffic profile simulating real-world network patterns with weighted distribution.

  • HTTP 40% / DNS 20%
  • TCP 15% / ICMP 10%
  • UDP 10% / App 5%
Use this mode
Protocol

Test a specific protocol with full parameter control. Choose TCP, UDP, ICMP, HTTP, DNS, FTP, SSH, or SMTP.

  • Single protocol focus
  • Full rate control
  • 8 protocols available
Use this mode
Stealth

Low-and-slow attack simulation with randomized timing. Tests detection of subtle, slow patterns.

  • Max 5 pps rate
  • Slow SYN scan
  • Randomized timing
Use this mode
Custom

Define your own traffic patterns via YAML configuration. Full control over generator types, rates, and weights.

  • YAML configuration
  • Multiple generators
  • Weighted distribution
Use CLI: trafficgoat custom -c config.yaml
CLI Usage
sudo trafficgoat stress   -t <target> -d 60 -r 500
sudo trafficgoat scan     -t <target> -p 1-1024
sudo trafficgoat mixed    -t <target> -r 200 -d 120
sudo trafficgoat protocol -t <target> --protocol icmp -d 30
sudo trafficgoat stealth  -t <target> -d 300
sudo trafficgoat custom   -t <target> -c config.yaml
